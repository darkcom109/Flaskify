Lesson 4: Static Files

In this lesson, you’ll learn how to use Flask’s static/ folder to serve CSS, JavaScript, and images.

Step 1: What Are Static Files?

Static files are assets that don’t change while your app is running — things like CSS (styling), JavaScript (interactivity), and images. Flask automatically looks for these files inside a folder called static/.

Step 2: Creating the Static Folder


        your_project/
        │── app.py
        │── templates/
        │    └── home.html
        └── static/
            ├── style.css
            └── logo.png

Create a folder named static in the same directory as app.py. Place your CSS, images, or JavaScript files inside this folder.

Step 3: Linking CSS in a Template


        <!DOCTYPE html>
        <html>
        <head>
            <title>My Flask App</title>
            <link rel="stylesheet" href="{{ "{{ url_for('static', filename='style.css') }}" }}">
        </head>
        <body>
            <h1>Hello, Flaskify!</h1>
        </body>
        </html>
    

        /* static/style.css */
        body {
            background-color: #f5f5f5;
            font-family: Arial, sans-serif;
            text-align: center;
        }

        h1 {
            color: #6a1b9a;
        }

Use url_for('static', filename='...') to safely link to files in your static/ folder. In this example, the CSS will style your background and heading.

Step 4: Adding an Image


        <img src="{{ "{{ url_for('static', filename='logo.png') }}" }}" alt="Flaskify Logo">

Place logo.png in the static/ folder. When you reload your page, Flask will serve the image from /static/logo.png.

Step 5: Adding JavaScript (Optional)


        <script src="{{ "{{ url_for('static', filename='app.js') }}" }}"></script>
    

        // static/app.js
        alert("Welcome to Flaskify!");

Just like CSS, you can load JavaScript files from the static/ folder. Try refreshing your page to see the alert.
