Lesson 1: Setup & Hello World

In this lesson, you'll install Flask and create your first "Hello World" app.

Step 1: Install Flask

pip install flask

Step 2: Create app.py


from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "Hello, Flaskify!"

Step 3: Run the App

flask run

Now open http://127.0.0.1:5000 in your browser. You should see "Hello, Flaskify!".
