Lesson 1: Setup & Hello World

In this lesson, you'll install Flask and create your first "Hello World" app.

Step 1: Install Flask


    pip install flask

This command uses pip, Python’s package manager, to download and install Flask into your environment. After running it, you’ll be able to import Flask in your Python scripts and start building web apps. If it doesn’t work, make sure Python and pip are properly installed and added to your system’s PATH.

Step 2: Create app.py


    from flask import Flask

    app = Flask(__name__)

    @app.route("/")
    def home():
        return "Hello, Flaskify!"

This script creates your first Flask application.

  • Flask(__name__) initializes your app.
  • @app.route("/") tells Flask which URL to listen to (in this case, the home page).
  • The home() function runs when you visit http://127.0.0.1:5000/ and returns "Hello, Flaskify!".

Step 3: Run the App


    flask run

Now open http://127.0.0.1:5000 in your browser. You should see "Hello, Flaskify!".
