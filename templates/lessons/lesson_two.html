Lesson 2: Routes & Views

In this lesson, you’ll learn how to create multiple pages in Flask by defining routes and views.

Step 1: What is a Route?


    @app.route("/")
    def home():
        return "This is the home page!"

A route is the URL pattern that maps to a function in your app. @app.route("/") means: “when someone visits the home page (/), run the home() function.”

Step 2: Adding Another Route


    @app.route("/about")
    def about():
        return "This is the about page!"

You can add as many routes as you like. In this case, visiting /about will run the about() function. Each route is tied to a view function, which returns the content shown in the browser.

Step 3: Multiple Routes in One App


    from flask import Flask

    app = Flask(__name__)

    @app.route("/")
    def home():
        return "Welcome to Flaskify!"

    @app.route("/about")
    def about():
        return "About Flaskify: Learn Flask, with Flask!"

    @app.route("/contact")
    def contact():
        return "Contact us at flaskify@example.com"

Now your app has three routes:

  • / → Home Page
  • /about → About Page
  • /contact → Contact Page
