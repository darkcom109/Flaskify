Lesson 0: Introduction

Welcome to Flaskify! Before diving into code, let’s talk about what Flask is and how this platform will help you learn it.

What is Flask?

Flask is a lightweight web framework for Python. It lets you turn regular Python code into a web application. With Flask, you can:

  • Create web pages with URLs and routes
  • Use templates to generate dynamic HTML
  • Handle forms, user input, and sessions
  • Connect to databases

Why Flask?

Flask is minimal but powerful. Unlike bigger frameworks (like Django), it doesn’t force you into a specific structure. This makes it perfect for beginners: you only learn what you need, when you need it.

What is Flaskify?

Flaskify is your personal learning platform — built with Flask, to teach you Flask. Each lesson will introduce a new concept, show you code examples, and give you space to practice.

What You’ll Learn

By the end of this course, you’ll know how to:

  • Set up and run a Flask project
  • Create routes and views
  • Use templates with Jinja
  • Add static files (CSS, images, JavaScript)
  • Handle forms and user input
  • Show flash messages for feedback
  • (Optional) Build a mini project with everything combined

Before You Start

Make sure you have:

  • Python installed (3.8+ recommended)
  • Basic knowledge of Python syntax
  • A code editor (like VS Code, PyCharm, or even IDLE if you’re brave)
